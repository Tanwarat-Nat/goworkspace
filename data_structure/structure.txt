data structure คือ
- การออกแบบและจัดการข้อมูลต่างๆและสามารถแก้ไขได้อย่างมีประสิทธิภาพ
- นอกจากเราเอาข้อมูลต่างๆเข้ามารวมกันแล้ว ยังบอกว่าข้อมูลแต่ละชุด แต่ละแบบ
ที่เราออกแบบโครงสร้างมามีความสัมพันธ์กันยังไงบ้างระหว่างกัน และมี operation
อะไรบ้างที่สามารถเข้าไปใช้งานกับมันได้

Usage
- data structure ต่างๆจะใช้ในการสร้าง abstract data type
abstract data type คือ พวก object, class 
มันจะมีข้อมูล และ set of operation ตรงนี้อยู่
