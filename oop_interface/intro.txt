นอกจากการเขียนโปรแกรม ในชีวิตจริงก็มีคำว่า interface เข้ามาเกี่ยวข้อง ยกตัวอย่างเช่น
เราอยู่ที่ร้านอาหาร เราบอกพนักงานว่าขอ น้ำอัดลมที่นึง สิ่งที่จะได้ อาจเป็น โค้ก สไปร์ เป๊ปซี่ ได้หมด
เพราะาทั้งหมดนี้เป็นน้ำอัดลม เพระาเราบอกไปว่าขอน้ำอัดลม คำว่าน้ำอัดลม คือ spec ของ interface
คือ contact ที่เราบอกกับพนักงานว่าเราต้องการสิ่งนี้นะ ต้องการน้ำอัดลมนะ เขาจะเอาน้ำหวานมาให้ก็ไม่ได้
requentment ที่ต้องมีก็คือ 
1.มันจะต้องเป็นน้ำอัดลม 
2.มันจะต้องมีรสชาติหวานๆ
ในการเขียนโปรแกรมก็จะมี concept ในเรื่องของ interface เข้ามาช่วย ก่อนอื่นที่จะมาเขียน interface
ของเราเอง ก่อนหน้านี้เราก็เคยใช้กันมาแล้วตั้งแต่ต้นเลยนั่นก็คือ function fmt.Println("")
("") ในตรงนี้ก็จะมีการใช้ interface อยูู่
