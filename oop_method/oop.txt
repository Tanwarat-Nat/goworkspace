oop หรือการเขียนโปรแกรมรูปแบบนึง โดยการคิดว่าสิ่งต่างๆคือ object(วัตถุ)
เลยทำให้มีผลต่อการเขียนโปรแกรมของเรา ซึ่งเราจะมองว่าสิ่งที่เรารับเข้ามาเป็น object
สิ่งที่เราจะ return ออกไปเป็น object สิ่งที่เราจะส่งต่อไป code อื่นๆเป็น object

oop คือ prigramming paradigm มันคือ วิธีการ classified แบ่งหมวดหมู่ programming
ต่างๆในโลก เป็นภาษาโปรแกรมคอมพิวเตอร์ว่ามันอยู่ใน หมวดไหน คลาสไหน อย่างบางภาษา
ยอมให้มีการเปลี่ยนแปลงค่าของ value ได้บางภาษาก็ไม่สามารถยอมให้เปลี่ยนแปลงได้
แบ่งเป็นหัวข้อใหญ่ๆ
1.imperative
 - procrdural
 - object oriented
2.declarative
 - function
 - logic
 - mathematical
 
สิ่งที่เราโฟกัสมีแค่นิดเดียว เป็น concept ที่ว่า จัดการกับปัญหาของเรา มองปัญหาตรงนั้น
แตกออกมาเป็น entity ย่อยๆแล้วก็มอง entity พวกนั้นเป็น object

ยกตัวอย่างเช่น เราจะมี Point X = 3, Y = 4 นึกถึงกราฟ 3 มิติ มีแกน X, Y
Point 3, 4 หมายความว่า X = 3, Y = 4 เราก้จะมองว่า Pointๆ นึงตรงนั้น
คือ object นึงในการเขียนโปรแกรมของเรา 
ดังนั้น object มันก็จะมีแบบ ข้อมูลอันนึงของมันและที่ต้องการ เราจะเรียกว่า value/variable
แต่ไม่แค่ object เท่านั้นมันยังมี method เข้ามาอีก

method คือ fuction อันนึงที่ทำการ operate กับ object อย่างเช่น
Lenght ถ้าจุดของมันคือ 3,4 ความยาวของมันก็คือ 5 (พีทาโกรัส)
MoveXTo ฟังชั่นย้าย X ไปที่จุดอื่น ย้ายจาก 3 -> 10 สุดท้ายได้จุดใหม่เป็น (10,4)
